#Polling Green Heron Rotor Controller for Azimuth data logging

# Introduction #

In and automated run, we'd like to be able to control the Green Heron Rotor controller from the software we write.  This way we can tweak the azimuth while hunting for open paths, and record the current azimuth during all events, such as echo returns.
